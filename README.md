# How Apps Work - Interactive Presentation

An exciting, interactive React presentation for secondary school students explaining how apps work from UI to API to Database.

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start the presentation
npm run dev
```

Then open http://localhost:5173 in your browser!

## 📖 Presentation Content

This 10-15 minute presentation includes:

1. **Title Slide** - Eye-catching introduction
2. **Hook** - Real-world Instagram example that teens use daily
3. **Frontend Layer** - HTML/CSS/JavaScript/React explained
4. **Code Example** - Real React code with fetch() API calls
5. **API Layer** - HTTP requests, JSON, REST endpoints
6. **Database Layer** - Tables, queries, PostgreSQL/MongoDB
7. **Complete Journey** - Posting a photo (UI → Cloud → DB)
8. **Reverse Journey** - Loading your feed (DB → Cloud → UI)
9. **Interactive Demo** - Live demonstration with animation
10. **Career Inspiration** - Real tech stack and salary potential

## 🎯 Target Audience

**15-17 year olds** who:
- Use apps like Instagram, Discord, Spotify daily
- Are curious about how technology works
- Might be considering computer science/software engineering
- Can handle real terminology (no "dumbing down")

## 🎮 How to Present

- **Navigate**: Use arrow keys (← →) or click the Previous/Next buttons
- **Spacebar**: Also advances to the next slide
- **Interactive Elements**: Some slides have clickable demos - try them!

## 🎨 Features

- Beautiful gradient backgrounds
- Smooth animations and transitions
- Interactive demonstrations
- Kid-friendly emojis and visuals
- Real-time examples students can interact with

## 💡 Key Concepts Explained

- **UI (User Interface)**: What students see and interact with
- **API (Application Programming Interface)**: The messenger between app and server
- **Database**: Where all data is stored safely
- **The Complete Flow**: How these layers work together

Perfect for engaging secondary school students with technology! 🎓
